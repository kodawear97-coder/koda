# Koda

Welcome to **Koda** - a repository for creative coding projects.

## Projects

### Ayan - Premium Food Photo Recreation

A premium landing page for food photo recreation services using HTML, Tailwind CSS, and Font Awesome.

**Features:**
- Responsive hero section with gradient backgrounds
- Floating animation effects
- Interactive buttons with hover effects
- Client testimonials and statistics
- Built with Tailwind CSS for styling

**View Live:** Open `index.html` in your browser to see the landing page.

## Technologies Used

- HTML5
- Tailwind CSS (via CDN)
- Font Awesome Icons
- Google Fonts (Montserrat & Inter)

## Getting Started

1. Clone this repository:
   ```bash
   git clone https://github.com/kodawear97-coder/koda.git
   ```

2. Open `index.html` in your browser to view the landing page.
